## 단어장 만들기

    def simple_tokennize(s:str):
      return re.findall(r"\b\w+\b", s.lower())

    MAX_VOCAB = 10000
    specials = ["<unk>", "<pad>"]

    counter = Counter()
    for _, txt in train_list:
      counter.update(simple_tokennize(txt))

    most_common = [w for w, _ in counter.most_common(MAX_VOCAB - len(specials))]
    itos = specials + most_common
    stoi = {w: i for i,w in enumerate(itos)}
    UNK_IDX = stoi["<unk>"]
    PAD_IDX = stoi["<pad>"]

- re.findall() -> 조건에 맞는 문자열들을 전부 찾아서 리스트로 변환
- s.lower() -> 문자열 s를 전부 소문자로 바꿔줘
- r"\b\w\b" -> 단어처럼 생긴 덩어리 찾기
- > \b -> 단어의 경계
  > 
  > \w+ -> 문자 숫자 밑줄이 1개이상 이어진것
  >

### 단어 사전 만들기 전체 흐름
1) 리뷰 문장을 단어로 나누는 함수 simple_tokenize를 만든다.
2) 훈련 데이터의 모든 리뷰를 돌면서 단어 빈도를 센다.
3) 가장 자주 나온 단어들만 뽑는다.
4) "<unk>", "<pad>" 같은 특별 토큰을 앞에 붙인다.
5) 단어↔숫자 변환용 사전 itos, stoi를 만든다.

    label_to_int = {"neg" : 0, "pos" : 1}

    def text_pipeline(x:str):
