# gradient - based - optimization

- 출력 = relu(dot(W,input)+b)
- W 와 b는 텐서
- W(가중치)와 b 네트워크가 훈련데이터를 통해 학습한 정보 담고 있다.
- 이 두개의 값을 작은 랜덤 값으로 초기화
- 이후 가충치들은 훈련되면서 점진적으로 조정

### - 훈련 루프 과정
- Step 1: 훈련 샘플 x와 정답 y_true를 배치로 가져온다

- Step 2: x를 네트워크에 넣어 예측값 y_pred를 얻는다 (forward pass)

- Step 3: y_pred와 y_true의 차이를 나타내는 손실(loss)을 계산한다

- Step 4: 손실을 조금 줄이는 방향으로 모든 가중치를 업데이트한다

- > 여기서 어려운 부분은 step4이다. 어떤 가중치를 얼마나 어떻게 바꿔야할지 어렵다
  >
  > 그래서 나온게 경사하강법(gradient descent)
  >
  > 손실(loss)를 가중치에 대해 미분한 값(gradient)를 구하고 그 반대방향으로 이동
  >
  > 손실을 줄일 수 있다.

### - 경사하강법

- 연속이고 매끄러운 함수  𝑓(𝑥 = 𝑦
- 함수는 연속적이기 때문에 f(x+ϵx) = y+ϵy
- 또한 함수가 매끄럽기 때문에 ϵx 작고 특정 점 p근처에서 기울기 a를 가지는 선형함수로 근사 가능
- > f(x+ϵx) = y + a⋅ϵx
-> x를 미분값의 반대방향으로 움직여서 조금만 이동 시키면 f(x) 값을 줄일 수 있

### - 텐서 연산의 미분
- 여러 차원 입력(텐서)을 가지는 함주에 대한 미분
- 다차원 공간에서의 함수가 어떻게 변하는지(표현의 기울기) 를 나타낸다

#### - 예시
- y_pred = dot(W,x)
- loss_value = loss(y_pred, y_true) -> 여기서 x와 y_true는 고정된 값
- 결국 loss_vlaue = f(W)
- > 기울기 구하기
  >
  > grad(loss_value, W) -> W로 미분 -> W에서의 미분값


-> 결국 우린 loss를 줄이기위해 
-> gradient의 반대방향으로 이동


### - 훈련 과정(Stochastic Gradient Descent) -> 데이터를 조금식 랜던하게 뽑아서(batch) 학습 -> 확률적
1) 데이터 일부 랜덤추출(batch)
2) 예측값 y_pred 계산
3) loss 계산
4) loss함수에서 W로 미분
5) 위에 값 반대로 파라미터 업데이트


### - backpropagation(역전파)
- 신경망의 구조는 여러 층(layer)들이 연쇄적으로 연결된 구조
#### - forward 계산
        - 데이터 -> 결과 -> loss계산
#### - backward 계산
        - gradient 계산 -> 뒤 레이어로 가면서 각각의 가중치 업데이
        - 여기서 쓰이는 계산 법은 chain rule
#### forward = 결과만들기
#### backward = 책임 따지기

#### - 전체 gradient 값들 곱해주기

### - 정리
-  학습이란, 주어진 훈련 데이터 샘플과 그에 대응하는 타깃에 대해 손실 함수(loss function)를 최소화하는 모델 파라미터의 조합을 찾는 과정이다.
- 앞으로 자주 등장하게 될 두 가지 중요한 개념은 손실(loss)과 옵티마이저(optimizer)이다. 이 두 가지는 네트워크에 데이터를 입력하기 전에 반드시 정의해야 한다.
