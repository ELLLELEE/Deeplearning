# PyTorch
### - PyTorch의 특징
1) 텐서 라이브러리 -> GPU에서 가속 연산을 수행할 수 있음
2) 자동 미분 엔진 autograd 제공해줌
3) 효율적인 구성 요소들(학습된 모델, 소실함수, 최적화 알고리즘)

### - 텐서 이해하기
- 텐서는 벡터와 행렬을 더 높은 차원으로 일반화한 수학적 개념
- 텐서는 차원수를 나타내는 차수(order or rank)
- 스칼라  2 -> rank0 , 벡터  -> rank1 , 행렬 -> rank2
- 계산적인 관점에서 텐서는 데이터 컨네이터 역할
- ex) 다차원 데이터를 담고 있으며, 각 차원은 서로다른 특징을 나타냄

### - data loader
-> 
    from torch.utils.data import Dataset
    
    class ToyDataset(Dataset):
    
    def __init__(self, X, y):
    
        self.features = X
        
        self.labels = y
    def __getitem__(self, index):
    
        one_x = self.features[index]
        
        one_y = self.labels[index]
        
        return one_x, one_y
    def __len__(self):
    
        return self.labels.shape[0]
        
    train_ds = ToyDataset(X_train, y_train)
    
    test_ds = ToyDataset(X_test, y_test)

  > 3) __len__ 메서드
  >
